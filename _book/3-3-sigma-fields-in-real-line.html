Counterexamples of Probability

3.3 Sigma-fields in real line

We next consider \(\sigma\)-fields when \(\Omega\) is the uncountable set \(\mathbb{R}\) (or an interval). Again, any useful \(\sigma\)-field must contain each singleton \(\omega\).

Definition 2.6 (Co-finite and co-countable) The set \(A\in\Omega\) is called co-finite set if its complement \(A^C\) is finnite. Further, the set \(A\in\Omega\) is called a co-countable set if \(A^C\) is countable.

Let \(\mathcal{F}_{1}\) consist of the finite and co-finite subsets of \(\Omega\). Then, \(\mathcal{F}_{1}\) is a field. It is a \(\sigma\)-field if and only if \(\Omega\) is finite. Let \(\mathcal{F}_{2}\) consist of the countable and the co-countable subsets of \(\Omega\). Then it is easy to check that \(\mathcal{F}_{2}\) is a \(\sigma\)-field.

Exercise 2.1 Prove that the collection of countable and co-countable sets is itself a \(\sigma\)-field.

Example 2.9 (A sigma-field of subsets of sample space need not contain all subsets of sample space) Suppose that \(\Omega\) is uncountable. Then \(\Omega\) contains a subset \(A\) such that \(A\) and \(A^C\) are both uncountable. This show that in general a \(\sigma\)-field of \(\Omega\) need not contain all subsets of \(\Omega\) and need not be closed under the formation of arbitrary uncountable unions.

Example 2.10 (Sigma-field of countable and co-countable sets: too small) Suppose that \(\Omega\) is uncountable set like \(\mathbb{R}\) or \([0,1]\). If we want \(\mathcal{F}\) to include each singleton \(\omega\), then \(\mathcal{F}\) must contain each countable subset \(A\). It must also contain the complement of each countable set. Therefore, any \(\sigma\)-field containing the singletons must contain all countable and co-countable sets. Unfortunately, this \(\sigma\)-field is not large enough to be useful. For instance, if \(\Omega=[0,1]\), then we are not allowed to consider events like \([0,\frac{1}{2}]\) because it is neither countable nor co-countable.
